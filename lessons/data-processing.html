Data Processing

Introduction

Data are raw facts, like numbers, words, or pictures. Information is data that has been processed so it becomes meaningful and useful.
Data processing is the process of taking raw data, doing something to it, and turning it into information that helps us make decisions.

Stages of Data Processing

  • Input: Collecting or entering data, e.g., pupils’ scores entered into a spreadsheet.
  • Processing: Sorting, calculating, or analysing data, e.g., finding total scores or averages.
  • Output: Giving the processed data in a useful form, e.g., charts, reports, or printed lists.
  • Storage: Saving information for future use, e.g., on a computer, USB, or cloud.
  • Feedback: Using the output to improve future data collection or processing.

Why Data Processing is Important

  • It helps people make better decisions by turning raw data into information.
  • It allows computers to handle large amounts of data quickly and accurately.
  • It helps us understand and organise information from everyday life, like school results.
  • Correct data input is important — wrong input leads to wrong output (“Garbage in, garbage out”).
