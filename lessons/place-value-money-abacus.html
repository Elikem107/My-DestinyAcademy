Place Value with money and abacus

Learning Objectives

  • Identify the value of digits in numbers using money and abacus.
  • Represent numbers with cedis and pesewas to understand place value.
  • Use an abacus to show hundreds, tens, and units.
  • Write numbers in expanded form using money and abacus.

Lesson Content

Place Value and Money: Money can be used to understand place value. For example, ₵325 means 3 hundreds (₵100 × 3 = ₵300), 2 tens (₵10 × 2 = ₵20), and 5 ones (₵1 × 5 = ₵5). So ₵325 = ₵300 + ₵20 + ₵5.

Place Value and Abacus: An abacus helps us see the value of each digit. Each rod of the abacus represents a place value — hundreds, tens, and ones. Beads on each rod represent the number of items in that place.

In an abacus, there are:

  • 3 beads on the hundreds rod → 300
  • 2 beads on the tens rod → 20
  • 5 beads on the ones rod → 5

That means the number represented is 325.

Class Activities

  • Use real or paper money to show the value of ₵245.
  • Draw an abacus and represent the number 478.
  • Write 523 in expanded form using money and abacus.
  • Work in pairs to compare two amounts of money and explain which is greater using place value.

Assessment / Exercises

  • 1. What does each digit in ₵472 represent?
  • 2. How many tens are there in ₵320?
  • 3. Represent ₵153 on an abacus.
  • 4. Write ₵406 in expanded form using money values.
  • 5. If there are 2 beads on the hundreds rod, 5 beads on the tens rod, and 3 on the ones rod — what number is shown?
  • 6. Explain how money helps you understand place value.
  • 7. Which is greater: ₵452 or ₵425? Explain using place value.
